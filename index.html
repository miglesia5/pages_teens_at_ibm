Este material fue preparado solo con la intencion de compartir, co crear y colaborar desde una perspectiva educativa.

1. Intro

GitHub es un servicio de alojamiento/hosting de sitio estático que toma archivos HTML, CSS y JavaScript directamente desde un repositorio en GitHub, opcionalmente ejecuta los archivos y los compila publicamente en un sitio web.

Por default se puede alojar el sitio en el dominio github.io

pero tambien se puede crear un dominio personalizado. Este modulo no cubre como crear un dominio personalizado pero Para obtener más información, consulta Usar un dominio personalizado con Páginas de GitHub".
